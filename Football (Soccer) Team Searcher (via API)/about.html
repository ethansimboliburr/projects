About Football Team Info

Website Purpose

Football Team Info is a simple and interactive web application designed to help users search for any professional football team and instantly view detailed information, including upcoming games, past results, and full player rosters. All information is fetched dynamically using TheSportsDB public sports API.

Features

  • Search for football teams by name
  • View upcoming scheduled matches and event times
  • Explore past games and final scores
  • View team roster with player images, positions, and nationalities
  • Responsive modern UI and animated cover page

Project Details

This project was created for a Web Development course to demonstrate skills such as external API integration, DOM manipulation, asynchronous JavaScript, multi-page website structure, and styling with CSS. It follows clean coding practices, including external files, organized folder structure, comments, and validated HTML.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript (ES6)
  • TheSportsDB API
